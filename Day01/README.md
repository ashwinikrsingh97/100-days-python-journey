## Notes & Reflections

**What was easy:**
> Printing and taking user input felt straightforward. The logic of combining two strings made sense right away.

**What was tricky:**
> Remembering that `input()` always returns a string — and being careful with spaces in concatenation.

**How this applies to production support / BFSI work:**
> `input()` and `print()` are the foundation of interactive CLI scripts — similar to how shell scripts prompt operators for inputs during incident response or manual batch runs.
